### Faturamento e Precificação

Relatório de Custo e Uso da AWS.

Os Relatórios de Custo e Uso da AWS (AWS CUR) contêm o conjunto mais abrangente de dados de custo e uso disponível. Você pode usar os Relatórios de Custo e Uso para publicar seus relatórios de faturamento AWS em um bucket do Amazon Simple Storage Service (Amazon S3) que seja seu proprietário.

AWS Cost Explorer.

O AWS Cost Explorer possui uma interface fácil de usar que permite visualizar, entender e gerenciar seus custos e uso da AWS ao longo do tempo. Ele não salva dados em um bucket Amazon S3.

### Conceitos de Nuvem

### Segurança

Criar usuários individuais de IAM

Você deve criar usuários individuais de IAM em vez de compartilhar contas de usuário.

Conceda o mínimo privilégio

É uma boa prática conceder o menor privilégio necessário para que os usuários possam desempenhar seu papel profissional.

Exclua a conta raiz

Você não pode excluir a conta, então isso não é uma boa prática.

Habilitar autenticação multifator (MFA)

É uma boa prática habilitar a MFA para a conta raiz (e para todas as contas).

Criar chaves de acesso para acesso programático

Você não deve criar chaves de acesso. Se tiver, certifique-se de que eles estejam bem trancados.

### Tecnologia

A escalabilidade é melhorada por meio de escalonamento por botão

Com RDS, você pode escalar verticalmente com o apertar de um botão.

Alta disponibilidade e recuperação de desastres são fáceis de implementar

Alta disponibilidade e recuperação de desastres são simples de implementar por meio de Read Replicas e Multi-AZ.

A escalabilidade é melhorada com o Amazon EC2 Auto Scaling

Você não pode usar Auto Scaling para escalar uma instância de banco de dados.

AWS Storage Gateway

Um gateway de arquivos AWS Storage Gateway fornece um servidor de arquivos virtual local, que permite armazenar e recuperar arquivos como objetos no Amazon S3. Um cache local fornece acesso de baixa latência aos dados usados recentemente.

Amazon EFS

O Amazon EFS pode ser montado usando interfaces de sistema de arquivos a partir de servidores locais. No entanto, ele não oferece cache local.

Amazon S3 One Acesso Zonalmente Infrequente

O S3 One Zone-IA é ideal para clientes que desejam uma opção de menor custo para dados acessados com pouca frequência, mas que não exigem a disponibilidade e resiliência do S3 Standard ou do S3 Standard-IA.

Glaciar Amazon S3

Esta classe de armazenamento é mais adequada para casos de uso arquivísticos e leva de minutos a horas para restaurar dados.

Configuração AWS

AWS Config é um serviço que permite avaliar, auditar e avaliar as configurações dos seus recursos AWS. O Config monitora e registra continuamente suas configurações de recursos AWS e permite automatizar a avaliação das configurações registradas em relação às desejadas.

Gerente de Sistemas AWS

O AWS Systems Manager oferece visibilidade e controle da sua infraestrutura na AWS. O Gerenciador de Sistemas integra-se com o AWS Config para gerenciamento de configuração.

AWS Trusted Advisor

O AWS Trusted Advisor verifica o uso de serviço que é superior a 80% do limite de serviço. Há 50 verificações de limite de serviço incluídas em todos os planos de apoio.

Amazon Inspector

O Amazon Inspector é um serviço automatizado de avaliação de segurança que ajuda a melhorar a segurança e a conformidade dos aplicativos implantados na AWS.
