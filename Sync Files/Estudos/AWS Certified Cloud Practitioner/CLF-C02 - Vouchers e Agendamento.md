## 🔔 Lembrete 4 — Pesquisar vouchers e descontos

**Data:** 04/06/2026 (Qui) · 10h | ⏱️ Tempo estimado: 30 min

> [!info] Primeiro passo (2 min)
> Abra o site da Compass no portal de benefícios (ou fale com o RH via Teams) e pesquise se existe parceria AWS Training. **Faça isso antes de qualquer outra busca** — é a maior chance de desconto.

### O que verificar — passo a passo

- [ ] Verificar se a **Compass UOL tem parceria AWS Training Partner** que dê vouchers para colaboradores
  - Canal: RH ou Deivison via Teams
  - A Compass é parceira AWS — é comum ter acesso a vouchers ou reembolso de certificação
- [ ] Verificar no portal de benefícios da Compass se há reembolso de certificações técnicas
- [ ] Checar o **AWS re/Start** ou programas de bolsa para profissionais de cloud
  - [aws.amazon.com/training/restart](https://aws.amazon.com/training/restart/)
- [ ] Verificar se o **Wellington ou alguém do time já fez a prova** — podem ter dicas de como conseguiram desconto
- [ ] Como último recurso: Udemy frequentemente vende cursos com voucher incluso (~R$ 50–80 na promoção)

### Preços de referência (sem desconto)

| Canal | Preço |
|---|---|
| Pearson VUE (padrão) | ~USD 100 (≈ R$ 550) |
| PSI Exams | ~USD 100 |
| Com voucher 50% (pós-reprovação) | ~USD 50 |
| Com parceria empresa | Pode ser grátis ou reembolsável |

> [!tip] Dica
> Se a empresa reembolsa, agende pelo Pearson VUE normalmente e guarde o comprovante de pagamento + certificado para solicitar o reembolso. O processo costuma ser simples via RH.

## ✅ Lembrete 8 — Agendar a prova (concluído)

**Feito:** prova confirmada para **22/08/2026 (Sábado)** — não 01/08 como o plano original previa; agendamento acabou saindo para uma data mais distante.

### Checklist de agendamento

- [x] Ter em mãos o voucher ou código de desconto (se conseguiu)
- [x] Acessar o [Pearson VUE — AWS](https://home.pearsonvue.com/aws)
- [x] Fazer login (ou criar conta se não tiver)
- [x] Selecionar **AWS Certified Cloud Practitioner (CLF-C02)**
- [x] Escolher centro de teste presencial
- [x] Data confirmada: **22/08/2026 (sábado)**
- [x] Confirmar e pagar (aplicar voucher se tiver)
- [ ] Anotar o número de confirmação da prova aqui: `________`
- [ ] Salvar o e-mail de confirmação

### O que levar no dia da prova

- Documento original com foto (RG ou CNH) — **obrigatório**
- Um segundo documento de identificação
- Chegar **30 minutos antes** do horário agendado
- Sem eletrônicos pessoais na sala (ficam no armário)
- Água pode entrar dependendo do centro — confirme antes

> [!warning] Atenção
> O Pearson VUE exige que o **nome no cadastro seja idêntico ao documento**. Verifique antes de confirmar o agendamento.
