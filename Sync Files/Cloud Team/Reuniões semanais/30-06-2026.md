## 📌 Glossário técnico desta reunião

| Termo | O que é |
|---|---|
| **Medâne** | Esteira/pipeline interna usada para deploy de Lambda. Limitação: só tem flag genérica "usa VPC" sem controle granular de subnet. |
| **VPCE (VPC Endpoint)** | Endpoint privado dentro da VPC para acessar serviços AWS sem passar pela internet. Algumas contas só têm VPCE, outras têm VPC completa com múltiplas subnets. |
| **Gateway Endpoint** | Tipo de VPC Endpoint gratuito para S3 e DynamoDB. Recomendação: toda VPC deveria ter — verificar contas antigas. |
| **Transit Gateway** | Serviço AWS que centraliza comunicação entre múltiplas VPCs e contas. Custo menor que alternativas como VPC Peering ponto a ponto. |
| **VPC Reachability Analyzer** | Ferramenta AWS para diagnosticar problemas de conectividade entre recursos numa VPC. Evento AWS mostrou uso com IA. |
| **Kiro** | IDE/ferramenta da AWS com IA integrada para troubleshooting. Pode ser aberto como terminal dentro da console AWS ou como app desktop. Uso baseado em tokens. |
| **DevOps Guru** | Serviço AWS de IA para troubleshooting operacional. Integra com outros serviços AWS e tem expertise nativa do ecossistema. |
| **Armazenamento dinâmico (Cloud SQL)** | Feature do GCP que aumenta automaticamente o disco da instância quando estoura (ex: +50GB, +100GB). Gera custo descontrolado se não monitorado. |
| **EOL (End of Life)** | Fim da vida útil de versões de banco de dados. Quando entra em EOL, Google cobra garantia estendida (custo adicional). |
| **Shaping (Cloud SQL)** | Processo de redimensionar instâncias de banco para tamanho adequado ao uso real. Ex: máquina com 2TB alocados usando 200GB. |
| **Vacuum (PostgreSQL)** | Processo de limpeza de espaço em disco no PostgreSQL — remove tuplas mortas e recupera espaço. Parte da avaliação que DBAs devem fazer. |
| **Catálogo (Spectra)** | Arquivo YAML que define serviços, seus ambientes, tecnologias e tags de busca. O orquestrador usa para rotear perguntas ao agente especializado correto. |
| **Prometheus Exporter (sidecar)** | Container que roda ao lado do serviço principal e expõe métricas para o Prometheus. Spectra usa as métricas do exporter para análise profunda (ex: CouchDB). |
| **synapse-docs** | Repositório no GitHub da empresa que versiona o Second Brain compartilhado do time. Base de conhecimento em Markdown integrada com Obsidian. |
| **Modo clássico (RabbitMQ)** | Tipo de fila no RabbitMQ que não suporta parada graceful para rolagem de instância. Causa problema na atualização do Amazon MQ. |
| **Matias e Cláudio** | Representantes da Rio/Cloud (Google) que vêm ao escritório quinta-feira para workshop. |
| **Wendel** | Responsável por monitoramento — vai acionar Lucas durante férias do Mateus para acompanhar plano de status. |
