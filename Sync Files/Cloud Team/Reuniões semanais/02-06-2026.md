## 📌 Glossário técnico desta reunião

| Termo | O que é |
|---|---|
| **BigQuery Reservation** | Modelo de capacidade reservada (slots) no BigQuery. Paga-se valor fixo mensal independente do uso. Se subutilizado, é desperdício puro. |
| **Efeito chicote (BQ)** | Quando se reduz a reserva, o custo de BQ on-demand sobe compensando — e vice-versa. Precisa calibrar. |
| **External Secrets** | Operador Kubernetes que sincroniza secrets de provedores externos (GCP Secret Manager, AWS Secrets Manager) para dentro do cluster. |
| **ServicePoint** | Portal interno (Sem Parar/Corpay) para requisição de acessos e provisionamento de grupos. |
| **PCI DSS** | Payment Card Industry Data Security Standard — certificação de segurança para empresas que processam dados de cartão. Renova anualmente. |
| **Stra** | Nome do projeto de AI Agents do Wellington. Visão: agentes especializados (SRE, Architect, FinOps) com MCPs compartilhados. |
| **MCP** | Model Context Protocol — padrão para conectar agentes de IA a ferramentas/sistemas externos (clusters, clouds, APIs). |
| **Spectra** | Nome planejado para o agente orquestrador do projeto Stra — recebe demandas e direciona para o agente correto. |
| **SES** | Simple Email Service — serviço AWS de envio de e-mails transacionais. Alternativa ao SMTP para os relatórios automatizados. |
| **Cowork** | Funcionalidade do Claude (Anthropic) para execução de código e automações — Wellington usou para prototipar o relatório AWS. |
| **Kong** | API Gateway open-source usado no cluster Gringo. Imagem antiga, sem atualização conhecida — risco de segurança para PCI. |
| **Freeze** | Período (outubro-novembro) onde mudanças em produção são restringidas por causa da Black Friday e preparação de janeiro. |
| **Google ADK** | Agent Development Kit — framework do Google para desenvolvimento de agentes de IA multiagentes. Sugestão do time para o projeto Stra. |
