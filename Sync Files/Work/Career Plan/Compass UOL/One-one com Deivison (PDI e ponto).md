## 📈 PDI — Plano de Desenvolvimento Individual

### O que é
Seu plano de evolução como profissional — **não é do Deivison, é seu**. Você define o que quer aprender e como vai comprovar.

### Estrutura recomendada por Deivison

```
PDI do Felipe
├── 1. Capacitações obrigatórias (Compass + Sem Parar)
├── 2. Hard Skills
│   └── ex: Azure, Terraform, certificação AWS
└── 3. Soft Skills
    └── ex: comunicação, senso de urgência, liderança
```

### Como montar cada atividade

| Campo | O que preencher |
|---|---|
| **O quê** | Objetivo claro ("Especializar em Terraform no Azure") |
| **Como** | Ações concretas (plataforma, curso, horas) |
| **Comprovação** | Como provar que concluiu (certificado, prints de progresso, módulo funcional) |
| **Prazo** | Factível — distribua ao longo do ano |
| **Prazo** | Curto: ~2 meses / Médio: ~3 meses / Longo: 4–6 meses |

> Leve o PDI para o Wellington também — ele provavelmente vai te apoiar com mentoria no lado técnico. Mostrar que você quer crescer conta ponto no cliente.
