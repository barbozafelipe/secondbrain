# CTASK0134547 — Inserir Variável de Ambiente no Web App Copilot (PRD)

---

## 📋 Descrição Original

> O recurso que deve receber as novas variáveis é: https://portal.azure.com/#@fleetcorbr.onmicrosoft.com/resource/subscriptions/b1ba0975-b0e9-4aa4-838a-cdb4b0c9e058/resourceGroups/stp-dig-rg-copilot-prd/providers/Microsoft.Web/sites/stp-dig-app-copilot-prd/appServices
> 
> A variável abaixo deve ser incluída em nosso recurso.
> 
> VARIAVEIS:
> 
> `RESUME_AGENT_URL`: \<entrar em contato com a equipe\>
> 
> Caso necessário, entrar em contato com a equipe de desenvolvimento do produto para editar as variáveis de ambiente manualmente no recurso.

---

## 🧠 Briefing da Tarefa

**O que precisa ser feito:** Adicionar **1 variável de ambiente** (`RESUME_AGENT_URL`) no Web App `stp-dig-app-copilot-prd` na Azure.

**Onde vou atuar:** Portal Azure → Subscription DIGITAL-PROD → RG `stp-dig-rg-copilot-prd` → Web App `stp-dig-app-copilot-prd` → Configuration → Application Settings.

**Principal cuidado:** O valor da variável **não está no chamado** — está marcado como "entrar em contato com a equipe". Você precisa obter esse valor com o time de desenvolvimento **antes** de aplicar. Não inventar valor.

**Como validar no final:** Confirmar que a variável aparece em Application Settings com o valor correto e que o Web App reiniciou sem erros após o save.
