Para a implementação de iniciativas de IA no B2B, precisamos criar um resource group e um recurso Azure AI Foundry para provisionar LLMs que serão utilizados para testes e homologações de pilotos. 

Este resource group poderá ser criado dentro da subscription DIGITAL-NPROD, mas será necessário um tagueamento com o centro de custo, pois todo custo deverá ser repassado para o Centro de Custos 100423 mensalmente.

O nome do RG pode ser: stp-dig-rg-aiagentsb2b-nprd

O nome do recurso AI Foundry pode ser: stp-dig-cog-aiagentsb2b-nprd

Os LLMs que devem ser provisionados são:
1. gpt-4.1 
2. gpt-4.1-mini
3. gpt-5.5 

Após a criação dos recursos, precisaremos compartilhar a chave com fernando.paiva@corpay.com.br , que está como ponto focal nos projetos. Estes recursos serão utilizados apenas para desenvolvimento e homologação. Eventuais mudanças para produção serão exigirão a criação




Ação:

Criei o RG manualmente na console do Azure, criei uma nova pasta no terraform (stp-dig-rg-aiagentsb2b-nprd), dentro dela criei os arquivos: resource_group.tf, provider.tf e  openai.tf

resource_group.tf:
module "rg" {

  source = "../../MODULES/resource_groups"

  

  resource_group_name = "stp-dig-rg-aiagentsb2b-nprd"

  location            = "brazilsouth"

  tags = {

    Projeto     = "AIHUB"

    CostCenter  = "100423"

  }

}



provider.tf:
provider "azurerm" {

  features {}

  subscription_id = "36df8ac5-dab6-4301-9cbf-97aa398ba021"

}

  

terraform {

  backend "azurerm" {

    storage_account_name = "pltidtsttfstate"

    resource_group_name  = "plt-idt-rg"

    container_name       = "tfsstate"

    key                  = "DIGITAL-NPROD/stp-dig-rg-aiagentsb2b-nprd/terraform.tfstate"

    subscription_id      = "8fc13250-5fbc-47a3-b05f-b0a0cc93b4b9"

    tenant_id            = "e710eef2-4915-4eba-8fbe-5fd8583a44f8"

  }

}



openai.tf:
resource "azurerm_cognitive_account" "oai-01" {

  name                          = "stp-dig-cog-aiagentsb2b-nprd"

  resource_group_name           = module.rg.resource_group_name

  custom_subdomain_name         = "stp-dig-cog-aiagentsb2b-nprd"

  location                      = "eastus2"

  kind                          = "OpenAI"

  public_network_access_enabled = true

  sku_name                      = "S0"

}

  

resource "azurerm_cognitive_deployment" "gpt41" {

  name                 = "gpt-4-1"

  cognitive_account_id = azurerm_cognitive_account.oai-01.id

  

  model {

    format  = "OpenAI"

    name    = "gpt-4.1"

    version = "2025-04-14"

  }

  

  sku {

    name     = "Standard"

    capacity = 10

  }

}

  

resource "azurerm_cognitive_deployment" "gpt41mini" {

  name                 = "gpt-4-1-mini"

  cognitive_account_id = azurerm_cognitive_account.oai-01.id

  

  model {

    format  = "OpenAI"

    name    = "gpt-4.1-mini"

    version = "2025-04-14"

  }

  

  sku {

    name     = "Standard"

    capacity = 10

  }

}

  

resource "azurerm_cognitive_deployment" "gpt55" {
