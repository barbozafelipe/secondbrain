## 📌 Glossário técnico desta reunião

| Termo | O que é |
|---|---|
| **DB Migration** | Scripts SQL que rodam via CI/CD para criar campos e automatizar alterações de banco de dados. Usam Secrets do GCP para credenciais — ponto de atenção na migração. |
| **Equinix Fabric / Megaport** | Serviços de interconexão de redes que permitem criar links dedicados entre on-premises e cloud. Sem Parar já usa essa tecnologia para outros links. |
| **MonoRepo** | Repositório único que contém múltiplos serviços/aplicações. 3 serviços do MonoRepo ainda não foram migrados para External Secrets. |
| **Startup Script** | Script que executa quando uma VM inicia no GCP. No Kong, fazia provisionamento, configuração e start do serviço — estava com bugs que causaram incidente. |
| **B2E** | Empresa/aplicação de back-office que vai rodar no RKE interno. Processa arquivos do ambiente Carvalt (P-Server). |
| **Guardrails** | Regras de segurança implementadas nos agentes de IA para impedir ações perigosas (ex: listar conteúdo de Secrets). |
| **ADR** | Architecture Decision Record — documento que registra decisões arquiteturais e seu contexto. Wellington usando no Spectra. |
| **Spec-Driven Development** | Abordagem onde specs/especificações guiam o desenvolvimento. Wellington cria specs e os agentes implementam features baseados nelas. |
| **Harbor** | Registro privado de imagens de container. Spectra usa Harbor para armazenar suas imagens. |
| **N8N** | Ferramenta de automação de workflows (similar ao Zapier). Já homologada na Sem Parar. Wellington planeja integrar com Spectra para triagem de incidentes. |
| **Foundry** | Plataforma de IA homologada na empresa (junto com ChatGPT, Claude, Copilot). |
| **Equinix Fabric** | Plataforma de interconexão — diferente de Megaport, mas mesmo propósito. Empresa usa para provisionar links com cloud providers. |
| **Darcimara** | Gestora do time de dados da Afinz/Carvalt — marcou reunião com Wellington mas time dela não participa das reuniões de quarta. |
| **Maracanã** | Contato/gestor no Olho no Carro — Rafael Humberto acionou para dar mais tração ao projeto de migração. |
