## 📌 Glossário técnico desta reunião

| Termo | O que é |
|---|---|
| **OnePrice / FreeFire** | Serviços/sistemas que precisam de conectividade com o novo cluster GKE Standard. Ponto de bloqueio atual na subida do cluster. |
| **Kong** | API Gateway open-source usado no cluster Gringo (GCP). Decisão de mantê-lo em vez de migrar para Ingress nativo do GKE. |
| **Network Policy** | Recurso nativo do Kubernetes que permite controlar tráfego de rede entre pods/namespaces. Usado como estratégia de segregação para PCI. |
| **Reloader** | Plugin/controller Kubernetes que monitora alterações em Secrets e ConfigMaps e dispara rollout automático nos Deployments afetados. |
| **External Secrets Operator** | Operador Kubernetes que sincroniza secrets de provedores externos (AWS Secrets Manager, GCP Secret Manager) para dentro do cluster automaticamente. |
| **AWS Secrets Manager** | Serviço AWS para armazenamento e rotação de segredos. Nova fonte de verdade para os clusters após migração (substitui GCP Secret Manager). |
| **Change** | Processo formal de gestão de mudança (ITIL) — precisa de aprovação antes de executar alterações em produção. |
| **Rollout** | Processo de reiniciar/atualizar pods de forma controlada no Kubernetes (rolling update). Necessário quando Secrets ou ConfigMaps mudam. |
| **K6 Operator** | Operador Kubernetes para o K6 (ferramenta de load testing da Grafana Labs). Identificado como não utilizado — não será migrado. |
| **Cash** | Ferramenta de FinOps do Google Cloud — relatórios de savings e recomendações. Wellington quer fazer POC e comparar com solução atual. |
| **Rio/Cloud** | Parceiro/representante do Google Cloud na América Latina. Argentino que vem fazer workshop de novas funcionalidades. |
| **Caio Medice** | Account Manager do Google para a conta Gringo — diferente do Caio Cruvinel (contato técnico). |
| **Matin / RFS** | Processo interno de requisição formal — Wellington exigiu que SI abra um antes de solicitar mudanças no ambiente. |
| **ConfigMap** | Recurso Kubernetes para armazenar configurações não-sensíveis. Wellington propôs substituir o banco PostgreSQL do Kong por ConfigMap. |
| **Autopilot (limitação)** | Modo gerenciado do GKE que não permite criar node pools separados nem configurar redes isoladas — limitação para PCI. |
