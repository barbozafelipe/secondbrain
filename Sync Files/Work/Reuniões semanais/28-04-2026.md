## 📌 Glossário técnico desta reunião

| Termo | O que é |
|---|---|
| **Transit Gateway** | Serviço AWS que interconecta múltiplas VPCs e redes on-premises |
| **Fiserv** | Empresa de tecnologia financeira — processa integração com Mastercard |
| **mTLS** | Mutual TLS — autenticação dupla entre cliente e servidor via certificado |
| **IAP** | Identity-Aware Proxy — acesso seguro a recursos GCP sem VPN |
| **HAProxy** | Load balancer de alta disponibilidade (Zapay) |
| **M7g / M5** | Famílias de instâncias AWS — M7g usa ARM (Graviton), M5 usa x86 |
| **Backstage** | Portal de desenvolvedor open-source (Spotify) — o time de DevOps está construindo um |
| **ImageStream** | Conceito OpenShift de versionamento de imagens — não existe no Kubernetes padrão |
