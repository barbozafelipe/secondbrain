**O que aconteceu na segunda-feira:**
- Google fez recall geral de instâncias Spot na região brasileira
- **26 worker nodes** derrubados de uma vez
- Pods ficaram em `Pending` — node selector dos scrapers estava como `required`, não permitindo subir em instâncias provisionadas
- João Pedro gastou ~1h30 de troubleshooting

**Correção aplicada:**
```yaml
# Antes — Spot obrigatório (pods ficavam em Pending sem Spot disponível)
nodeSelector: { preemptible: "true" }  # required

# Depois — Spot preferencial, mas sobe em qualquer nó se não houver
affinity:
  nodeAffinity:
    preferredDuringSchedulingIgnoredDuringExecution: [...]
```

**Status**: ✅ Resolvido por João Pedro.

## 📌 Glossário técnico desta reunião

| Termo | O que é |
|---|---|
| **Spot Instance** | VM com desconto que pode ser interrompida pela cloud a qualquer momento |
| **Node Selector `required`** | K8s: pod só sobe no tipo de nó especificado — sem alternativa |
| **Node Selector `preferred`** | K8s: prefere o tipo especificado, mas sobe em outro se necessário |
| **GKE Autopilot** | Modalidade do GKE onde o Google gerencia os nodes automaticamente |
| **HPA** | Horizontal Pod Autoscaler — escala pods com base em métricas |
| **CA (Certificate Authority)** | Autoridade que emite e valida certificados TLS internos |
| **JKS / Truststore** | Formato Java de armazenamento de certs — apps configuram a CA aqui para validar TLS |
| **Kafka** | Sistema de mensageria — apps Java se conectam via TLS e precisam reconhecer a CA |
| **Cloud NAT** | GCP: saída de tráfego de VPCs sem IP público — 1 VPC por gateway |
| **Imperva** | WAF/CDN externo — requer envolver SI para novas publicações |
| **Zscaler** | Proxy de segurança para acesso à internet nos notebooks corporativos |
| **Isaú** | SRE — ponto de contato para comunicar frentes de dev sobre rotação de CA |
| **Yasu** | Membro do time da Gringo — ajudou Thiago a criar o repositório de IaC |
| **mTLS** | Mutual TLS — autenticação dupla entre cliente e servidor via certificado |
| **CAPEX / OPEX** | Classificação financeira dos projetos no board para lançamento de horas |
